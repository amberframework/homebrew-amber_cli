# Homebrew Tap for Amber CLI

Homebrew formulae for the [Amber Framework](https://amberframework.org) CLI and LSP.

## Installation

```bash
brew tap crimsonknight/amber-cli
brew install amber-cli
```

This installs two binaries:

- **`amber`** — CLI for generating and managing Amber projects
- **`amber-lsp`** — Language Server for Claude Code integration

## Updating

```bash
brew update
brew upgrade amber-cli
```

## Uninstall

```bash
brew uninstall amber-cli
brew untap crimsonknight/amber-cli
```
